# Agentforce 360 Decision Tree

Interactive decision tree tool for qualifying Agentforce 360 opportunities and generating personalized transformation roadmaps.

## Features

- ✅ 12 strategic discovery questions
- ✅ Personalized recommendations based on industry, role, and use case
- ✅ ROI calculator with cost estimates
- ✅ Customer success story links
- ✅ Interactive timeline and roadmap
- ✅ Salesforce-branded design
- ✅ Mobile responsive

## Built With

- React 18
- Vite
- Lucide Icons
- Tailwind CSS (CDN)
- Salesforce Sans Font

## Deployment

This project is configured for easy deployment to Vercel.

## Local Development

```bash
npm install
npm run dev
```

Open http://localhost:5173 to view it in your browser.

## Build

```bash
npm run build
```
